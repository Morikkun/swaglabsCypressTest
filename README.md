Hey there, how is it going? This repository waas created to E2E test the workflow of an e-commerce application using Saucelabs

## Requirements
-- Node 24.13.0 or higher
-- Git 2.53.0 or higher
-- Cypress 15.8.2 or higher

## Installation
-- Download Node from [Node official page](https://nodejs.org/pt-br/download)
-- Download Git from [Git official page](https://git-scm.com/install/windows)
-- Download Cypress by running the `npm install cypress --save-dev` on your command prompt

## Tests
-- `cy:open` runs the tests using the Cypress App
